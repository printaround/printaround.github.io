JOHN'S MATH

C#, Console

Categories: Math

Technically, this is my very first digital game. I decided to add it for the sake of nostalgia and because I considered it quite playable, even by its current standards. The game is a console dialogue with an assistant, where you have to solve simple mathematical examples, but the main point of the game is how you interact with the assistant, since it will depend on what kind of ending you get (like there are only 3 of them). I will not attach the source code for obvious reasons ... however, I decided to leave it in its original form, so I do not advise leaving an empty line at the moments where you need to enter a number, otherwise the program will throw an exception.
